Importing the source code into ECLIPSE
--------------------------------------

1. File > Import > Android: Existing Android Code Into Workspace
2. Browse. Choose Zellar folder.
3. Check `Copy Projects Into Workspace`

Android Libraries used
----------------------
Importing these libraries are required to run the source code. Except for ListViewAnimation and Picasso (uses jar file, see libs folder)

- HoloEverywhere (https://github.com/Prototik/HoloEverywhere)
- google-play-service (Download via Android SDK Manager)
- ListViewAnimation (https://github.com/nhaarman/ListViewAnimations)
- Picasso (https://github.com/square/picasso)
